## 1）我们到底在数什么？

我们要数的是：有多少段连续区间 `nums[l..r]`，它们的和等于 `k`。

比如 `l=2, r=5` 就是一段从 2 到 5 的连续子数组。

---

## 2）先引入一个“累加到当前位置”的量：前缀和 pre

我们边走边累加：

* `pre` 表示：从数组开头一直加到当前这个位置的总和

例如 nums = `[2, -1, 3]`

* 走到 2：`pre=2`
* 再加 -1：`pre=1`
* 再加 3：`pre=4`

所以 `pre` 是“到目前为止总共加了多少”。

---

## 3）关键：子数组和 = 两个前缀和的差

假设我们走到了位置 r，此时 `pre` 就是 `nums[0..r]` 的和。

如果我们想要某个子数组 `nums[l..r]` 的和：

* `nums[0..r]` 的和是 `pre[r]`
* `nums[0..l-1]` 的和是 `pre[l-1]`

那中间那段 `nums[l..r]` 就是：

`pre[r] - pre[l-1]`

这句话你可以用“整段减去前面那段，剩下中间那段”来理解。

---

## 4）把“等于 k”改写成“找一个以前出现过的前缀和”

我们要的是：

`pre[r] - pre[l-1] = k`

移项：

`pre[l-1] = pre[r] - k`

**这句话非常关键，我用口语翻译：**

> 我现在走到 r，总和是 `pre[r]`。
> 想找一段以 r 结尾、和为 k 的子数组，就需要以前某个位置的前缀和刚好等于 `pre[r] - k`。
> 以前这样的前缀和出现过几次，就有几段子数组。

---

## 5）所以我们要做一件事：记录“以前每种前缀和出现了多少次”

这就是：

```cpp
unordered_map<int,int> cnt;
```

`cnt[s]` 表示：前缀和等于 `s` 的情况，之前出现了多少次。

---

## 6）为什么要先写 `cnt[0] = 1` ？

这是最常见卡点，我慢慢说。

`cnt[0]=1` 表示：在你还没开始遍历数组之前，存在一个“空前缀”，它的和是 0，而且出现 1 次。

它用来处理这种情况：**子数组从 0 开始**。

举例：`nums = [3, ...]`，`k=3`

走到第一个数 3：

* `pre = 3`
* 你想找 `pre - k = 0`
* 如果 `cnt[0]=1`，你就能找到这段子数组 `[3]`（从 0 到 0）

如果没有 `cnt[0]=1`，你会漏掉所有“从 0 开始就满足”的子数组。

---

## 7）现在逐行走代码（真的慢慢走）

你的代码：

```cpp
unordered_map<int,int> cnt;
cnt[0] = 1;
int pre = 0, ans = 0;

for (int x : nums) {
    pre += x;
    if (cnt.count(pre - k)) ans += cnt[pre - k];
    cnt[pre]++;
}
return ans;
```

我把循环每次做的事情拆成 3 步：

### 第一步：更新当前前缀和

```cpp
pre += x;
```

意思：我走到当前位置，把当前数加进来。

### 第二步：统计以“当前位置结尾”的答案数量

```cpp
ans += cnt[pre - k];
```

意思：以前出现过多少次 `pre-k`，就说明有多少段子数组以我结尾，和为 k。

### 第三步：把当前前缀和存起来，给后面用

```cpp
cnt[pre]++;
```

## 8）我再用一个“带负数”的例子跑一遍（更能看懂“次数”的意义）

nums = `[1, -1, 0]`，k = 0

初始化：

* `cnt = {0:1}`
* `pre=0, ans=0`

### 位置0：x=1

1. `pre=1`
2. 查 `pre-k = 1`：`cnt[1]` 不存在 → 加 0
3. `cnt[1]++` → `cnt={0:1, 1:1}`

ans=0

### 位置1：x=-1

1. `pre=0`
2. 查 `pre-k = 0`：`cnt[0]=1` → `ans += 1`

   * 对应子数组 `[1,-1]`
3. `cnt[0]++` → `cnt={0:2, 1:1}`

ans=1

### 位置2：x=0

1. `pre=0`
2. 查 `pre-k = 0`：现在 `cnt[0]=2` → `ans += 2`

   * 对应两个子数组：

     * `[0]`（位置2自己）
     * `[1,-1,0]`（从0到2）
3. `cnt[0]++` → `cnt={0:3, 1:1}`

最终 ans=3 ✅

你看到了吗：**同一个前缀和 0 出现多次，会产生多个答案**，所以必须记“次数”。

---

## 9）你现在只需要记住一句话

> 我每到一个位置 r，就把前缀和算出来 `pre`。
> 以前有多少次出现过 `pre-k`，就有多少个子数组以 r 结尾且和为 k。
